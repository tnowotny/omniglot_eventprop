# omniglot_eventprop
Investigations of eventprop learning on the omniglot dataset

In this repository, we are investigating how to recognise symbols from the [omniglot dataset](https://github.com/brendenlake/omniglot) using the eventprop [1] methods in [ml_genn](https://github.com/genn-team/ml_genn).


References
1. Wunderlich, T.C., Pehle, C. Event-based backpropagation can compute exact gradients for spiking neural networks. Sci Rep 11, 12829 (2021). https://doi.org/10.1038/s41598-021-91786-z
